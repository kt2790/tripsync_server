# tripsync_server

## 1. 프로젝트 소개
---
여행 계획 작성 및 공유 서비스를 제공하는 TripSync 앱의 백엔드 프로젝트 입니다.
[프론트엔드 깃허브](https://github.com/NBCAndroid15/TripSync)

## 2. 사용 기술
---
- Spring Boot 3.3.2
- Spring Web MVC
- Spring Cache
- Spring REST Docs
- Spring Data JPA
- Query DSL
